### React Quick Start

Ensure Node.js (`node` and `npm`) is installed: https://docs.npmjs.com/downloading-and-installing-node-js-and-npm
    -   Install via official site\
        OR
    -   with homebrew:
        brew install npm

Install dependencies and start the React app:

    sh

    npm install

    npm start
